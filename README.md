cpp-collections
===============

### How to ?


1.  Clone the repository

    $ git clone https://github.com/Giraudux/cpp-collections.git

2.  Compile and build the project

    $ make

or

    $ make clean && make

3.  Run the program

    $ ./build/parse -help
    $ ./build/parse -tree -print -most-frequent ./doc/lipsum.txt
